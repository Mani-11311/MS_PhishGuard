# Contributing to MS_PhishGuard

## How to contribute

- Fork and clone the repo.
- Create a new branch (`feature/my-feature`).
- Update/add your improvements.
- Push your branch and open a pull request.
- Describe your change, include screenshots when relevant.

## Code Style

- Follow existing HTML/CSS/JS conventions.
- Keep UX simple and visually accessible.
- Document your hacks and decisions.

## Reporting Bugs

- Open a issue labelled “bug.”
- Describe reproduction steps.

## Feature Requests

- Open a issue, explain your need.

## Code of Conduct

Be respectful. No spam/trolling. Collaborative and open for all.
