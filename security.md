# Security Policy

## Supported Versions

We take security seriously! Please report all suspected security vulnerabilities.

| Version | Supported   |
| ------- | ----------- |
| 2.x     | ✅          |
| 1.x     | ❌          |

## Reporting Security Issues

If you discover a potential security issue in MS_PhishGuard:

- Please **do not disclose publicly**.
- Email [manichaturvedi1719@gmail.com] directly, or create a ‘security’ issue (github private).
- We will investigate promptly.

## Community Best Practices

- Never share sensitive URLs publicly.
- Always update dependencies for patched CSS/JS libraries (e.g., Tailwind/Admin templates).

## Responsible Disclosure

You’ll always be credited unless you wish to be anonymous.
